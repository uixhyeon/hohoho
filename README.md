# hohoho

## 📚 Code Archive 프로젝트

리퀴드텍스트 스타일의 코드 아카이브 - 코드와 메모를 시각적으로 연결하여 학습 내용을 정리하는 React 앱

---

## 🚀 Git Bash에서 실행하는 순서

### 1. 프로젝트 클론하기
```bash
# 프로젝트 클론
git clone https://github.com/uixhyeon/hohoho.git

# 프로젝트 폴더로 이동
cd hohoho
```

### 2. code-archive로 이동
```bash
cd code-archive
```

### 3. 패키지 설치
```bash
npm install
```

### 4. 개발 서버 실행
```bash
npm run dev
```

### 5. 브라우저에서 열기
- 브라우저에서 `http://localhost:5173` 접속
